# ICE + Japan-Fit スコアリング基準

## ICE Score 概要

| 要素           | 意味                        | 範囲 |
| -------------- | --------------------------- | :--: |
| **I**mpact     | 実装時のビジネス影響力      | 1-10 |
| **C**onfidence | 成功確信度 (根拠信頼度)     | 1-10 |
| **E**ffort     | 実装容易度 (容易なほど高い) | 1-10 |

---

## Impact 点数基準

### 収益貢献度 (40%)

| 点数 | 基準                             | 例示                    |
| :--: | -------------------------------- | ----------------------- |
|  10  | 直接収益創出 (サブスク/アプリ内) | プレミアム専用機能      |
|  8   | 転換率 10%+ 向上予想             | ペイウォール最適化      |
|  6   | 転換率 5-10% 向上予想            | 価値実演機能            |
|  4   | 間接収益貢献                     | リテンション → 収益連結 |
|  2   | 収益連結不明確                   | 便利機能                |

### リテンション貢献度 (40%)

| 点数 | 基準                   | 例示                 |
| :--: | ---------------------- | -------------------- |
|  10  | D1 リテンション核心    | オンボーディング改善 |
|  8   | D7 リテンション核心    | 習慣形成機能         |
|  6   | D30 リテンション貢献   | 長期動機付与         |
|  4   | リテンション間接貢献   | コンテンツ多様性     |
|  2   | リテンション連結不明確 | 付加機能             |

### 差別化 (20%)

| 点数 | 基準           | 例示          |
| :--: | -------------- | ------------- |
|  10  | 市場唯一       | 革新的AI機能  |
|  7   | 競合社対比優位 | より良い実装  |
|  4   | 競合社と同等   | Must-have機能 |
|  1   | 差別化なし     | 基本機能      |

## Japan-Fit Score

### 評価基準

| 点数 | 基準                           | 例示                           |
| :--: | ------------------------------ | ------------------------------ |
|  10  | 日本市場専用/最適化            | 敬語学習、J-POPコンテンツ      |
|  8   | 日本ユーザー強い選好           | 洗練されたUX、ポイントシステム |
|  6   | グローバル + 日本適合          | 音声認識、AIチューター         |
|  4   | グローバル共通、日本特殊性なし | 基本学習機能                   |
|  2   | 日本市場不適合                 | 西洋文化コンテンツ             |

> Japan-Fitは単一点数(1-10)で評価。別途加算点なく以下の最終公式でボーナスとして反映。

---

## 最終点数計算 (v3 加重平均公式)

> **v3 変更**: 既存 `(I×C)/E` 公式は範囲が不確定(0.1~100)であり、正規化公式が恒等関数(`x/10×10=x`)であるバグがあった。
> v3は加重平均に転換して自動範囲保証(1.0~10.0) + Japan-Fitボーナスを統合。

```
# Step 1: Base ICE (加重平均)
Base ICE = (Impact × 0.4) + (Confidence × 0.3) + (Effort × 0.3)
  → 範囲: 自動 1.0 ~ 10.0

# Step 2: Japan Bonus (5点超過分のみボーナス)
Japan Bonus = max(0, (Japan-Fit - 5)) × 0.2
  → 範囲: 0 ~ 1.0
  → Japan-Fit ≤ 5: ボーナス 0 (Japan-First 戦略反映)
  → Japan-Fit = 10: ボーナス 1.0

# Step 3: Final Score
Final Score = clamp(Base ICE + Japan Bonus, 0, 10)
```

### 公式利点

| 特性             | 説明                                                    |
| ---------------- | ------------------------------------------------------- |
| **範囲保証**     | Base ICEが自動で1.0~10.0 (正規化不要)                   |
| **Japan-First**  | Japan-Fit > 5のみボーナス (5以下は中立、ペナルティなし) |
| **AI暗算可能**   | 加重合算 → 単純な四則演算                               |
| **既存Tier互換** | 8.0+ Tier 1基準維持                                     |

### 点数別推奨アクション

| 点数範囲 |  Tier  | 推奨アクション              |
| :------: | :----: | --------------------------- |
|   8.0+   | Tier 1 | 即時 feature-architect 実行 |
| 6.0-7.9  | Tier 2 | 追加調査後決定              |
| 4.0-5.9  | Tier 3 | バックログ登録、MVP後検討   |
|  < 4.0   | Tier 4 | 却下または長期保留          |

### マイグレーションノート

- **既存候補 (v1/v2)**: `ice_score` フィールドは `ice_score_v1` として保存
- **新規候補**: v3 加重平均公式で `ice_score` 計算
- Tier 分類基準(8.0/6.0/4.0)は同一に維持

## 例示

### 例示 1: 発音チャレンジモード

| 項目       | 点数 | 根拠                                       |
| ---------- | :--: | ------------------------------------------ |
| Impact     |  7   | D7 リテンション貢献 (ゲーミフィケーション) |
| Confidence |  8   | Duolingo, Speak 成功事例                   |
| Effort     |  6   | 既存音声認識活用可能                       |
| Japan-Fit  |  8   | 日本人発音矯正ニーズ高い                   |

```
Base ICE = (7 × 0.4) + (8 × 0.3) + (6 × 0.3)
         = 2.8 + 2.4 + 1.8 = 7.0

Japan Bonus = max(0, (8 - 5)) × 0.2 = 3 × 0.2 = 0.6

Final Score = clamp(7.0 + 0.6, 0, 10) = 7.6
```

**結果**: Tier 2, 追加調査後決定

### 例示 2: ソーシャル学習コミュニティ

| 項目       | 点数 | 根拠                         |
| ---------- | :--: | ---------------------------- |
| Impact     |  6   | リテンション間接貢献         |
| Confidence |  5   | 成功/失敗事例混在            |
| Effort     |  3   | モデレーションコスト高い     |
| Japan-Fit  |  4   | 日本人コミュニティ参加度低い |

```
Base ICE = (6 × 0.4) + (5 × 0.3) + (3 × 0.3)
         = 2.4 + 1.5 + 0.9 = 4.8

Japan Bonus = max(0, (4 - 5)) × 0.2 = 0 (Japan-Fit ≤ 5)

Final Score = clamp(4.8 + 0, 0, 10) = 4.8
```

**結果**: Tier 3, バックログ登録。Japan-Fit 4はボーナスなし → Japan-First 戦略適合

### 例示 3: AI Error Pattern Tracker

| 項目       | 点数 | 根拠                             |
| ---------- | :--: | -------------------------------- |
| Impact     |  9   | D7+D30 リテンション核心 + 差別化 |
| Confidence |  8   | 5件リサーチ根拠、競合社分析      |
| Effort     |  7   | 既存AIパイプライン活用           |
| Japan-Fit  |  9   | L1(日本語) 転移エラー特化        |

```
Base ICE = (9 × 0.4) + (8 × 0.3) + (7 × 0.3)
         = 3.6 + 2.4 + 2.1 = 8.1

Japan Bonus = max(0, (9 - 5)) × 0.2 = 4 × 0.2 = 0.8

Final Score = clamp(8.1 + 0.8, 0, 10) = 8.9
```

**結果**: Tier 1, 即時 feature-architect 実行
