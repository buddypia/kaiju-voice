# コード最適化コマンド

## 実行内容

このコマンドは以下の最適化を自動的に実行します:

### 1. パフォーマンス最適化

- 不要な`setState`の削除および最適化
- `const`コンストラクタの適切な使用
- ウィジェット再ビルドの最小化
- 重い処理を`compute`関数に分離/移管

### 2. メモリ最適化

- 未使用importの削除
- メモリリーク検出および修正
- `dispose`メソッドの正しい実装
- `StreamSubscription`のクリーンアップ

### 3. 状態管理最適化

- 不要なProvider再ビルド防止
- `ref.watch` / `ref.read`の適切な使い分け
- `autoDispose`の活用
- `family` Providerの最適化

### 4. ビルド/レンダリング最適化

- ウィジェットツリーの分離および最適化
- `RepaintBoundary`の適切な配置
- 画像/アセットの最適化
- リストの仮想化（`ListView.builder`）

### 5. コード品質改善

- 未使用コードの削除
- 重複コードのリファクタリング
- ネーミング規則の統一
- 複雑なコードに日本語で詳細なコメントを追加/更新
